Choose:

```
$ brew install composer
$ composer install
```

Or:

```
$ # download https://getcomposer.org/download/1.3.2/composer.phar
$ php /path/to/composer.phar install
```

Set database configuration: `./vendor/database.ini`, whatever you want:

```
[mysql]
host = "localhost"
dbname = "dlxs"
user = "user"
password = "password"
```
